# Archived Prototype Source

This directory contains the previous Rust Defender implementation exactly as it
existed before the clean-slate red-label rewrite began.

Archived code is kept for reference only. Useful pieces can be copied or
translated into the new `src/` tree when they are compatible with the exact
arcade-core plan in `SPEC.md`.

Contents:

- `src/`: previous coarse-grid gameplay, rendering, audio, ROM table, and
  session implementation.
- `tests/`: previous integration tests.
- `examples/`: previous README media generator.
- `tools/`: extraction scripts used to derive checked-in assets and tables.
